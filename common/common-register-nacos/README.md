# 单机部署

- Linux/Unix/Mac

```shell
$ startup.sh -m standalone
# standalone代表着单机模式运行，非集群模式
```

- Windows

```shell
$ startup.cmd -m standalone
```

[可视化管理界面](http://localhost:8848/nacos)
